# FaceForm Channel Autoposter

Отдельный Telegram-бот/сервис для автоматических публикаций в канале FaceForm. Каждый запуск:

1. ищет свежую новость или тренд через OpenAI Web Search;
2. создаёт оригинальный русскоязычный пост;
3. проверяет базовые редакционные ограничения;
4. генерирует фирменную PNG-карточку без чужой фотографии;
5. публикует пост в Telegram-канал;
6. сохраняет использованные темы, чтобы уменьшить повторы;
7. завершает процесс — подходит для Railway Cron.

## Что публикуется

- мода и заметные тренды;
- волосы, причёски, борода и макияж;
- безопасный луксмаксинг без унижения внешности;
- уход без медицинской диагностики;
- beauty-tech и новости индустрии;
- практические разборы с указанием источника.

Сервис не копирует статьи целиком и не использует фотографии из новостей. Вместо этого создаётся собственная брендированная карточка.

## 1. Telegram

Можно использовать существующего FaceForm-бота — второй бот необязателен. Автопостер не включает polling и не меняет webhook, поэтому тот же токен может использоваться сервером Mini App и этим Cron-сервисом.

Добавьте бота администратором канала и выдайте право **публиковать сообщения**:

```text
Канал → Изменить → Администраторы → Добавить администратора
```

В `TELEGRAM_CHANNEL_ID` укажите публичный username с `@`, например:

```env
TELEGRAM_CHANNEL_ID=@faceform_ru
```

Для приватного канала нужен числовой `chat_id` вида `-100...`.

## 2. OpenAI

Создайте API-ключ и положите его в Railway как `OPENAI_API_KEY`. Этот сервис использует Responses API с инструментом Web Search.

Рекомендуемая модель:

```env
OPENAI_MODEL=gpt-5-mini
```

Использование API оплачивается отдельно от подписки ChatGPT.

## 3. Railway

Лучше добавить автопостер **отдельным сервисом в том же Railway-проекте**, чтобы не менять основной FaceForm.

### Переменные

Скопируйте `.env.example` в Variables и заполните:

```env
TELEGRAM_BOT_TOKEN=токен_бота
TELEGRAM_CHANNEL_ID=@username_канала
OPENAI_API_KEY=ключ_OpenAI
OPENAI_MODEL=gpt-5-mini
FACEFORM_BOT_URL=https://t.me/username_бота?startapp
CHANNEL_NAME=FaceForm | Разбор внешности
CHANNEL_USERNAME=@username_канала
STATE_FILE=/app/data/content-bot-state.json
DRY_RUN=false
SAMPLE_MODE=false
DISABLE_CARD=false
```

### Volume

Подключите Railway Volume к пути:

```text
/app/data
```

Без Volume бот продолжит работать, но после пересоздания контейнера может забыть ранее опубликованные темы.

### Cron

В сервисе откройте:

```text
Settings → Cron Schedule
```

Для двух публикаций в день — примерно в 11:00 и 19:00 по Москве:

```cron
0 8,16 * * *
```

Railway использует UTC. Один запуск создаёт один пост и завершает процесс.

## 4. Первый тест

Перед включением реальной публикации установите:

```env
DRY_RUN=true
SAMPLE_MODE=true
```

Запустите Deploy вручную. В логах появится демонстрационный текст, но в канал ничего не отправится.

Затем проверьте OpenAI без Telegram:

```env
DRY_RUN=true
SAMPLE_MODE=false
```

Когда текст устроит:

```env
DRY_RUN=false
SAMPLE_MODE=false
```

Запустите сервис вручную один раз. После успешного поста включите Cron Schedule.

## 5. Редакционная безопасность

Промпт запрещает:

- унижение внешности и веса;
- диагнозы по фотографии;
- обещания гарантированной привлекательности;
- операции, препараты и экстремальные диеты;
- копирование больших фрагментов источника;
- публикацию слухов из анонимных каналов;
- смешивание мужских и женских примеров там, где профиль важен.

Автоматическая публикация всё равно не даёт абсолютной гарантии качества. Первую неделю регулярно просматривайте канал и уточняйте редакционный промпт под свой стиль.
